# Patterns

Composition recipes — how primitives and atoms come together in real Pulse surfaces. Each pattern below is a canonical layout we reuse across routes.

- **List route** — skeleton → empty → staggered list → side-panel edit → undo toast.
- **KPI strip** — display font (Fraunces) + tabular-nums (JetBrains) in the same tile.
- **Form layout** — labeled grid in a card or side panel.
- **Status badges** — the five semantic badge slots we use across pipeline, payroll, pulse.
- **Labs surface** — how "NEW"-flagged experimental features look.

Copy these structures directly. Tokens are already wired — don't reach for hex.
