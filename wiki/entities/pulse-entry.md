# Pulse Entry

One anonymous vibe submission.

## Definition

A date, a vibe (amazing / good / meh / rough / awful), an optional anonymous note, and an optional tag (workload / team / tools / growth / other). **No employee id is ever attached.**

## Connected entities

- Aggregated to a [[Department]] and to the org for the [[Pulse]] heatmap.
- Cannot be tied back to an [[Employee]] — by design.

## Where it appears

[[Pulse]] · [[Dashboard]] (HR / manager rollup)
