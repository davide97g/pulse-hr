# Plan

A lightweight scope document attached to a [[Commessa]]. Team, tasks, FAQ, status. Used for the kickoff doc, not full project management.

## Definition

A title, an owner ([[Employee]]), a [[Commessa]] link, a status (draft / active / archived), a free-form body with sections.

## Connected entities

- Belongs to a [[Commessa]].
- Authored by an [[Employee]].
