# commessa

**Italian term.** Plural: *commesse*. Pronounced *kom-MES-sa*.

## English

Project code · billable engagement · job number.

## Context

In Italian agency, consultancy, and contracting culture, every billable engagement gets a *commessa* — a code (like `ACM-2025-01`) that hours, expenses, allocations, and invoices are pinned against. The commessa is the unit of accounting truth: if work happened off-commessa, finance can't bill it.

Pulse HR is built around this concept. Anglophone users can read "commessa" as "billable project" — but it's distinct enough (and ubiquitous enough in Italian SMBs) that the wiki keeps the original term.
