# Settings

Workspace-level and personal-level preferences. Theme, locale, notification cadence, role override (for [[Admin]]), birthday opt-in for [[Moments]].

## Who uses it

Everyone (their own settings). [[Admin]] for workspace-wide settings.

## Key entities

[[Employee]] (preferences live on the employee record).

## Notable behaviors

- **Theme switcher.** Pick from [[Role Themes]] — light, dark, employee, hr, manager, admin, finance.
- **Locale.** English / Italian. Italian unlocks [[busta paga]] / [[ferie]] / [[permesso]] terminology.
- **Workspace reset.** Wipe local-only state for [[Demo Mode]] and start over.
