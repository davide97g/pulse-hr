# Marketplace

The integrations index. Slack, Google Calendar, identity providers (SSO), accounting tools, payroll providers — anything that plugs into Pulse HR.

## Who uses it

[[Admin]] mostly. [[HR]] occasionally for HR-specific integrations.

## Key entities

(No HR-facing entity — internal config.)

## Notable behaviors

- Each integration has an install / configure / disconnect flow.
- Connected integrations surface in the relevant feature (e.g. a Google Calendar connection makes [[Calendar]] mirror the user's external calendar).
- Per [[Open Source Positioning]], integrations are configurable; the Pulse HR principle is "you can always export and disconnect."
