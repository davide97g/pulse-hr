# Keyboard-First

Pulse HR is designed so a keyboard-only user can navigate and act faster than a mouse user. A deliberate stance — most HR tools assume click-heavy UI; this one inverts.

## Core shortcuts

- `⌘K` / `Ctrl+K` → [[Command Palette]] (find anything).
- `⌘J` / `Ctrl+J` → [[Copilot]] (do anything).
- `⌘\` / `Ctrl+\` → toggle the sidebar.
- `?` → open the shortcut sheet.
- Per-route shortcuts (e.g. `n` to add a new record on a list page).

## Why

Customer-research signal: the people running [[HR]] / [[Finance]] / [[Manager]] desks all day are power users. Saving them three clicks per action across 40 actions a day adds up.
