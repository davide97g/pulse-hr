# Command Palette

Pulse HR's `⌘K` (or `Ctrl+K`) fuzzy finder. The fastest way to navigate.

## What it finds

- [[Employee]]s — by name, role, department, location.
- [[Commessa]]e — by code, name, client.
- [[Document]]s — by title.
- [[Settings]] entries.
- [[In-App Docs]] topics.
- Sidebar destinations (jump to any feature without clicking).

## Distinct from [[Copilot]]

The command palette **navigates and finds**. Copilot **does** — it executes actions like "log 4h" or "request leave."
